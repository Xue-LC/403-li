## 🚀 如何预览

### 方法 A：SSH 隧道（推荐）

**在你本地电脑运行**：
```bash
ssh -L 5173:localhost:5173 root@你的服务器 IP
```

**然后浏览器访问**：
```
http://localhost:5173
```

### 方法 B：临时开放端口

**在服务器上**：
```bash
# 临时开放 5173 端口
ufw allow 5173/tcp

# 访问
http://你的服务器 IP:5173

# 测试完后关闭
ufw deny 5173/tcp
```
